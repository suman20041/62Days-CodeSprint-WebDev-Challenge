# 🎨 Color Palette Extractor from Image

Upload an image and **extract dominant colors** automatically — copy values as **HEX**, **RGB**, or **HSL**. Built with HTML, CSS, JavaScript, and the **Canvas API**.

---

## ✨ Features

- **Image upload** with preview
- **Dominant color extraction** via Canvas pixel sampling
- **Color palette** displayed as swatch cards
- **One-click copy** for HEX, RGB, and HSL values
- **Drag-and-drop** image support
- **Loading and error states** for invalid files
- **Adjustable color count** (3–10 swatches)
- **Responsive** layout

---

## 🚀 How to Run

No installation needed.

1. Open `index.html` in any modern browser (Chrome, Firefox, Edge).
2. Upload or drag & drop an image.
3. View the palette and click **Copy** on any color format.

### Optional — local server

```bash
npx serve .
```
