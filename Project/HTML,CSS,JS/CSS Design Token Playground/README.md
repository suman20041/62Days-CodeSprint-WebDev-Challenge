# 🎨 CSS Design Token Playground

A beginner-friendly tool to **experiment with CSS design tokens** and see **live UI component updates** — built with HTML, CSS, and JavaScript.

Edit colors, spacing, and typography tokens on the left; watch buttons, cards, badges, and alerts update instantly on the right.

---

## ✨ Features

- **Token editor** for colors, spacing, and typography
- **Live component preview** — buttons, cards, badges, alerts
- **Real-time updates** using CSS custom properties
- **Copy tokens** — export generated `:root` CSS block
- **Reset to defaults** with one click
- **Responsive** split-panel layout

---

## 🚀 How to Run

No installation needed.

1. Open `index.html` in any modern browser (Chrome, Firefox, Edge).
2. Adjust tokens in the left panel.
3. Click **Copy Tokens** to export your design system values.

### Optional — local server

```bash
npx serve .
```
