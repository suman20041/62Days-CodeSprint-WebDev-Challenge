## Overview

This repository contains an official site/landing page for **SmartScraps**, an intelligent waste monitoring system concept designed to create cleaner, safer communities. The landing page outlines the vision and features of the waste monitoring platform, which transforms ordinary bins into smart data points for real-time monitoring and management.

---

## Features

*   **📊 Real-Time Monitoring:** Get live data on bin fill-levels, air quality (gas buildup), and flame presence transmitted via WiFi.
*   **🚨 Live Alerts & Notifications:** Receive instant notifications on web and mobile platforms when predefined thresholds are crossed, signaling an immediate need for service.
*   **↔️ Modular & Scalable System:** Our solution is scalable for individual bins or vast dumping zones, featuring rugged IP67-rated sensors built for extreme conditions.
*   **📍 Live Mapping:** An interactive map displays the real-time status and location of all users and smart bins/zones.
*   **🏢 Versatile Deployment:** Ideal for deployment in apartment complexes, educational institutions, corporate campuses, metro stations, and other public spaces.
*   **🌱 Environmental Safety:** Proactively identifies and helps eliminate hazardous waste, such as items that release toxic gases or are flammable.

---

## Tech Stack

This landing page is built with fundamental web technologies.

*   **HTML5:** For the structure and content of the page.
*   **CSS3:** For styling, layout (including Flexbox and Grid), and responsiveness. Custom properties (variables) are used for theming.
*   **JavaScript:** For dynamic features like the active navigation link highlighting on scroll.
*   **Google Fonts:** Used for custom typography ('Rubik Mono One' and 'Titillium Web').
