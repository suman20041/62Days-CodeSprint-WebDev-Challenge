# 📄 Markdown Resume Live Preview Builder

A beginner-friendly **Markdown resume editor** with **live preview** and **print/PDF-ready** output — built with HTML, CSS, and JavaScript.

Write your resume in Markdown on the left, see a formatted professional layout on the right, and print or save as PDF directly from the browser.

---

## ✨ Features

- **Split-screen layout** — Markdown editor + live preview side by side
- **Real-time rendering** — preview updates instantly as you type
- **Professional resume template** — clean typography and section styling
- **Print / PDF ready** — use browser Print → Save as PDF
- **Sample resume** — one-click starter template
- **Responsive UI** — works on desktop and mobile

---

## 🚀 How to Run

No installation needed.

1. Open `index.html` in any modern browser (Chrome, Firefox, Edge).
2. Edit the Markdown on the left or click **Load Sample**.
3. Click **Print / Save PDF** to export your resume.

### Optional — local server

```bash
npx serve .
```

Then visit `http://localhost:3000`
