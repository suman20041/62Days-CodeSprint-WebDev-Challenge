# ✅ Open Source PR Checklist Generator

Generate **open source contribution checklists** and **commit message templates** based on project type — HTML/CSS/JS, React, MERN, Node.js, or Documentation. Built with HTML, CSS, and JavaScript.

---

## ✨ Features

- **Project type selector** — HTML/CSS/JS, React, MERN, Node.js, Docs
- **Auto-generated checklist** tailored to each project type
- **Commit message templates** — feat, fix, docs, refactor
- **One-click copy** for checklist and templates
- **PR best practices tips** for beginners
- **Responsive** clean UI

---

## 🚀 How to Run

No installation needed.

1. Open `index.html` in any modern browser (Chrome, Firefox, Edge).
2. Select a project type and click **Generate**.
3. Copy the checklist or commit templates for your PR workflow.

### Optional — local server

```bash
npx serve .
```
