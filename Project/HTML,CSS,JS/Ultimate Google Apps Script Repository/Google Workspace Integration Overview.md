# 📜 Google Workspace Integration Overview

This section covers how to use Google Apps Script to connect and automate different Google Workspace applications.

**Google Apps Script** acts as the glue between services, allowing you to create powerful custom workflows. For example, you can automatically create a calendar event from a new row in a Google Sheet, or generate a Google Doc from a Google Form submission.

The power of Apps Script lies in its container-specific and standalone script capabilities, enabling seamless integration across the entire Google ecosystem.

---

### Core Subtopics:

- **Working with Google Docs**: Programmatically create, read, and edit Google Docs.
- **Working with Google Sheets**: Manipulate spreadsheets, read and write data, and create custom functions.
- **Working with Google Forms**: Build forms, respond to submissions, and manage form data.
- **Working with Google Calendar**: Create and manage events, calendars, and guest lists.

Each subtopic contains code examples within its respective folder.
