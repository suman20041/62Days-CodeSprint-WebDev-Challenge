# 🌐 Fundamentals Overview

This section explores the fundamental concepts of Google Apps Script. It covers everything from basic syntax to core services for automating your workflows in Google Workspace.

**Google Apps Script** is a cloud-based scripting language for light-weight application development in the Google Workspace platform. It is based on JavaScript, providing a familiar environment for developers.

---

### Core Subtopics:

- **Basic Syntax**: Learn about variables, data types, operators, and control flow in the context of Apps Script.
- **Triggers**: Discover how to run scripts automatically, based on time-driven events or actions in Google Workspace applications (like on form submit or document open).
- **Core Services**: Explore the built-in services that allow you to interact with user data and Google systems, such as `Logger`, `PropertiesService`, and `CacheService`.

Each subtopic contains code examples within its respective folder.
