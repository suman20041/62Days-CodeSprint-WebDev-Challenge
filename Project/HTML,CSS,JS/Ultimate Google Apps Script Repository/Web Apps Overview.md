# 🐘 Web Apps Overview

This section covers how to build and deploy web applications using Google Apps Script. You can create user interfaces that are accessible on the web, either privately within your domain or publicly.

**Apps Script's HTML Service** allows you to serve web pages that can interact with your server-side Apps Script functions, similar to other web development stacks but hosted entirely on Google's infrastructure.

---

### Core Subtopics:

- **HTML Service Basics**: Learn how to serve basic HTML, CSS, and client-side JavaScript.
- **Server-Client Communication**: Understand how to call server-side Apps Script functions from your client-side code using `google.script.run`.

Each subtopic contains code examples within its respective folder.
