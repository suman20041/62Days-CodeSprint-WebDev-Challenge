# ✊✋✌️ Rock Paper Scissors Game

A responsive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**. This project offers a clean, interactive, and engaging experience where players compete against the computer. The interface is fully responsive, providing a seamless experience across desktops, tablets, and mobile devices.

## 🎯 How to Play

1. Select **Rock**, **Paper**, or **Scissors**.
2. The computer randomly chooses its move.
3. The winner is determined using the standard game rules:

   * Rock beats Scissors
   * Scissors beats Paper
   * Paper beats Rock
4. The result is displayed instantly after each round.
