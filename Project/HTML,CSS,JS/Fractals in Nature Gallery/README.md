### 📜 Overview

This project is a multi-page static website built to display fascinating fractal patterns found in Plants, Geology, and Weather. It features a clean, dark-themed UI and interactive slideshows on each page. The homepage includes a combined gallery that cycles through all the images from the different categories.

---

### ✨ Features

*   **Multiple Galleries**: Separate, themed galleries for Plants, Geology, and Weather, plus a comprehensive gallery on the homepage.
*   **Automatic Slideshow**: Images in each gallery automatically transition every 4 seconds with a smooth fade effect.
*   **Manual Navigation**: Users can click the previous and next arrows to navigate through the images at their own pace.
*   **Responsive Design**: The layout adapts to different screen sizes for a great viewing experience on both desktop and mobile devices.
*   **Elegant UI**: A modern, dark-themed design with custom fonts (`Cinzel Decorative`) that enhances the visual appeal of the fractal images.

### 🚀 How To Run

To get a local copy up and running, follow these simple steps.

1.  **Clone the repository**

2.  **Navigate to the project directory**
    ```bash
    cd "Fractals in Nature Gallery"
    ```

3.  **Open `index.html`**
    
    Simply open the `index.html` file in your favorite web browser to launch the website.

    ```bash
    # On macOS
    open index.html

    # On Windows
    start index.html
    ```

- You may even add more images of your own to the sub-folders, and also add more sub-folders to this project!
