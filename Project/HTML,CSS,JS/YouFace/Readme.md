# 🎭 YouFace

YouFace is a fun web application that creates a humorous face based on the **first letter of your name** and your **selected gender**. After entering your name and submitting, the application extracts the first letter, generates a matching funny face, and displays it with the message:

> **"This is you!" 😄**

## ✨ Features

- 👤 Select your gender
- 📝 Enter your name
- 🔤 Automatically extracts the first letter
- 🎭 Generates a funny face based on your initial and gender
- ⚡ Instant result with a fun reveal
- 📱 Simple and responsive interface

## 🚀 How It Works

1. Select your gender.
2. Enter your name.
3. Click the **Submit** button.
4. The application extracts the first letter of your name.
5. A funny face is generated based on your gender and first letter.
6. The generated face is displayed along with the message **"It is you!"**
