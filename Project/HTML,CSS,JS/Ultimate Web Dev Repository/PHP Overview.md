# 🐘 PHP Overview

This section covers PHP, a popular server-side scripting language especially suited for web development. It can be used to create dynamic web pages, manage databases, and build entire web applications.

**PHP (Hypertext Preprocessor)** is widely used due to its simplicity, large community, and strong integration with databases like MySQL.

---

### Core Subtopics:

- **PHP Syntax**: Learn the basic syntax, variables, control structures, and functions in PHP.
- **Form Handling**: Understand how to collect and process data from HTML forms securely.
- **Database with MySQLi**: Explore how to connect to and interact with a MySQL database using the modern MySQLi extension.

Each subtopic contains code examples within its respective folder.
