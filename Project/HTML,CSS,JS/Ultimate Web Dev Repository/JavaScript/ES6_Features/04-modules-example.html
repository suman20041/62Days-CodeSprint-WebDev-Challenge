ES6+ Modules (Single-file demo)

Note: Real modules typically use multiple files and type="module". This repo’s viewer may not execute module loading like a bundler, so this file demonstrates module syntax conceptually inside comments.

Example (what you’d put in separate files)

// math.js
export function add(a, b) {
  return a + b;
}

// main.js
import { add } from './math.js';
console.log(add(2, 3));
