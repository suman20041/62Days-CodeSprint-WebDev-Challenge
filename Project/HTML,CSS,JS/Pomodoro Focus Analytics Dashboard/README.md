# 🍅 Pomodoro + Focus Analytics Dashboard

A **Pomodoro timer** with **session tracking** and a **focus analytics dashboard** — daily/weekly stats and charts. Built with HTML, CSS, JavaScript, **Chart.js**, and **localStorage**.

---

## ✨ Features

- **Pomodoro timer** — 25 min focus, 5 min short break, 15 min long break
- **Start, pause, reset, skip** controls
- **Session history** saved in `localStorage`
- **Daily analytics** — last 7 days session count & focus time charts
- **Summary stats** — total sessions, total focus time, today, best day
- **Recent sessions** list
- **Responsive** clean UI

---

## 🚀 How to Run

No installation needed.

1. Open `index.html` in any modern browser (Chrome, Firefox, Edge).
2. Start a focus session and complete it to log analytics.
3. View charts and stats update automatically.

### Optional — local server

```bash
npx serve .
```
