# Smart Habit Tracker

Educational MERN project for issue **#246**.

## Features
- JWT auth, habits CRUD, check-ins
- Timezone-safe streak engine
- Reminder rules
- Friend challenges + leaderboard
- Custom weekly heatmap

## Run

```bash
cd backend && cp .env.example .env && npm install && npm run dev
cd frontend && npm install && npm run dev
```

## Tech Stack
MongoDB · Express · React (Vite) · Node.js
