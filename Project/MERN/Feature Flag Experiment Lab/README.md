# Feature-Flag & Experiment Lab (LaunchDarkly-lite)

Educational MERN project for issue **#254**.

## Features
- Auth-protected flag management UI
- Projects + flags stored in MongoDB
- Percentage rollouts + segment targeting
- Audit log for configuration changes
- Live React demo SDK evaluating flags via Express

## Run
```bash
cd backend && cp .env.example .env && npm install && npm run dev
cd frontend && npm install && npm run dev
```
